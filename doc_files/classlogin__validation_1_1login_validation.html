Detailed Description

Class for validating login credentials.

This class defines a method for validating login credentials against a database.
It interacts with database connections to verify the provided credentials.

:param username: The username to be validated.
:type username: str
:param clean_pw: The clean (unhashed) password to be validated.
:type clean_pw: str

login_validation.loginValidation.validate_credentials ( self) 
Validates the provided login credentials.

This method validates the provided username and password against a database.
It connects to the database, retrieves the hashed password associated with the username,
and compares it with the provided clean password after hashing.
If the credentials are valid, it returns a list containing [True, user_id, accountType],
otherwise it returns [False, None, None].

:return: A list containing [is_valid, user_id, accountType].
:rtype: list
